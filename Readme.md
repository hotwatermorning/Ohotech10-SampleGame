# このプログラムは

["Ohotech 特盛 #10"](http://ohotech.connpass.com/event/7517/)(2014/08/30)で発表した内容のサンプルプログラムです

# ビルド方法

## 必須環境

* Visual Studio 2013
* Dxライブラリ
* Boost 1.56.0

## 環境設定

* Visual Studio 2013でソリューションを開き、プロパティーマネージャー画面から以下の２つのプロパティシートを変更する

 * Boost.propsのインクルードパス、リンクライブラリのパスで使用している$(BOOST_ROOT)をBoost 1.56.0をインストールしたディレクトリに変更する（もしくは環境変数に%BOOST_ROOT%を適切なパスで定義する）
	
 * DxLib.propsのインクルードパス、リンクライブラリのパスで使用しているDxライブラリのパスを、DxLibをインストールしたディレクトリに変更する
